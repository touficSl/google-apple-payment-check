# Google Apple Payment Check

A WPF Windows application to verify client subscription status on Apple App Store and Google Play Store.

## üìã Introduction

WPF C# Windows app designed to help users check if a client is subscribed to Apple or Google Store using a user-friendly GUI.

The only thing to do is to select between Google or Apple store and then fill the required fields for Google or Apple, and the app will check and return a view of all information about the current payment.

**Note:** Only Apple payment is available for now.

## ‚ú® Advantages

- **Field Discovery Helper**: Includes a "help to find this field" feature to assist users in locating required information
- **Comprehensive Payment View**: Displays a full view of the current payment details
- **User-Friendly Interface**: Simple GUI for easy subscription verification

## üöÄ Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/touficSl/google-apple-payment-check.git
   cd google-apple-payment-check
   ```

2. **Open in Visual Studio**
   - Open `Apple-Google-Payment-Check.sln` in Visual Studio

3. **Build the solution**
   - Restore NuGet packages
   - Build the solution (F6 or Ctrl+Shift+B)

4. **Run the application**
   - Press F5 or click the Start button in Visual Studio

## üí° How to Use

1. Launch the application
2. Select the store type (Apple or Google)
3. Fill in the required fields for the selected store
4. Use the "help to find this field" feature if needed
5. Click to verify the subscription
6. View the complete payment information displayed
